# Recursion_Problems 🚀

Welcome to the **Recursion_Problems** repository! This project contains a collection of fundamental algorithmic problems implemented in Java using recursion. It serves as a personal archive for practicing recursive logic, stack manipulation, and problem-solving.

## 📂 File Mapping & Implemented Problems

Here is the breakdown of the files currently in the repository:

| File Name | Description |
| :--- | :--- |
| `Practice_Problem_1.java` | Implementation of the Tower of Hanoi puzzle. |
| `Practice_Problem_2.java` | Logic to print a string in reverse using recursion. |
| `Practice_Problem_3.java` | Finding the first and last occurrence of an element in a string. |
| `Practice_Problem_4.java` | Checking if an array is sorted in strictly increasing order. |
