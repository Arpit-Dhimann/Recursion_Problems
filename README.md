## 💡 Key Learning Takeaways

*   **Base Case Mastery:** Learned when and how to stop recursion to avoid `StackOverflowError`.
*   **State Management:** Passing data effectively through recursive parameters (e.g., `res`, `idx`, `HashSet`).
*   **Optimization:** Understanding how to reduce time complexity and stack height using the **Divide and Conquer** approach.
